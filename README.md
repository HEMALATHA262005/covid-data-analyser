# 🦠 COVID-19 Data Analyser

A data analysis project built using Python, pandas and SQLite 
that analyses real-world COVID-19 data across 200+ countries.

## 📌 About the Project
This project loads the Our World in Data COVID-19 dataset, 
cleans it using pandas, stores it in a SQLite database, and 
runs SQL queries to generate meaningful insights.

## 🛠️ Technologies Used
- Python 3.x
- pandas (data cleaning & analysis)
- SQLite (database storage)
- SQL (data querying)

## 📊 Key Insights Generated
- Top 10 countries by total COVID-19 cases
- Top 10 countries by death rate %
- Top 10 most vaccinated countries
- Yearly global case trend (2020–2023)
- India specific statistics
